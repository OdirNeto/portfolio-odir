# Portfólio Online — Odir Pereira Borges Neto

Portfólio pessoal desenvolvido como Atividade Prática da disciplina
**Desenvolvimento Web** — UNINTER.

## 🛠️ Tecnologias

- HTML5
- CSS3 (puro, sem frameworks)
- JavaScript (puro, sem frameworks/bibliotecas)

## 📂 Estrutura

```
portfolio-odir/
├── index.html        (Sobre Mim - página inicial)
├── formacao.html     (Formação acadêmica e cursos)
├── portfolio.html    (Projetos)
├── contato.html      (Formulário de contato)
├── css/
│   └── estilo.css
├── js/
│   └── script.js
└── README.md
```

## ✅ Funcionalidades

- 4 páginas separadas, todas com menu de navegação fixo
- Design responsivo (desktop, tablet e mobile)
- Menu hamburguer dinâmico em mobile
- Alternância de tema claro/escuro (persistido em localStorage)
- Filtro dinâmico de projetos por categoria
- Formulário de contato com validação JavaScript completa e modal de confirmação

## 🚀 Como rodar localmente

Basta abrir o arquivo `index.html` em qualquer navegador moderno.
